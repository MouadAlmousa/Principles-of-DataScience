This dataset contains information on students’ demographics, education background, and exam performance across three subjects: Math, Reading and Writing.
 • gender is string describe student’s gender with two values: ['female' 'male']
 • race/ethnicity is string describe student’s ethnic group with four values: ['group B' 'group C' 'group A' 'group D' 'group E']
 • parental level of education is string describe highest education level attained by student’s parent with six values:: ['bachelor's degree' 'some college' 'master's degree' 'associate's degree' 'high school' 'some high school']
 • lunch is string describe type of lunch received with two values: ['standard' 'free/reduced']
 • test preparation course is string describe completion of test preparation course with two values : ['none' 'completed']
 • math score is an integer describe score in math exam between 0 and 100
 • reading score is an integer describe score in reading exambetween 0 and 100
 • writing score is an integer describe score in writing exam between 0 and 100
